# **App Name**: Command Center

## Core Features:

- Dashboard Overview: Display key performance metrics and trends visually, including market dominance, map pack control, indexed listings, and revenue impact, with a dynamic visibility trend chart.
- Keyword Position Tracking: Provide a detailed grid of keyword positions, showing individual keyword performance, rank changes, and search volumes over time.
- Interactive Data Visualization: Utilize charts (e.g., bar charts from Recharts) to represent data trends and metrics in an easily digestible graphical format.
- User Profile and Plan Display: Show the logged-in user's name and subscription plan in the application header for personalization.
- Data Fetching and Management: Integrate with a backend API to retrieve and manage real-time performance metrics and keyword data for the dashboard.

## Style Guidelines:

- A dark theme predominates, featuring a very dark background (e.g., #111826, #1C1A23) with subtle gray text (#888, #400). Interactive elements and data visualizations are highlighted with a vibrant blue (e.g., #2D6BFF).
- Uses clear, modern sans-serif fonts for all text elements. Headings and key metrics use bold and larger sizes, while descriptive text remains legible. A consistent font is used throughout for a professional appearance.
- Navigation links and other UI elements will use simple, clean line-based icons, consistent with a professional dashboard aesthetic. Icons should be monochromatic and blend seamlessly with the dark theme.
- Features a responsive, multi-column grid layout for displaying metric cards and a clear sidebar-main content structure for navigation and core features. Tables are used for detailed data display, and ample padding ensures a clutter-free interface.
- Subtle, smooth animations are incorporated, such as Framer Motion's `whileHover` effects on metric cards, to enhance user interaction and provide visual feedback without distraction.
